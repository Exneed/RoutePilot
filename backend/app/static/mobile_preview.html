🚀 Çalıştırma Komutları

# Dependencies
flutter pub get

# Android
flutter run

# iOS
flutter run -d iphone

# Web
flutter run -d chrome

# Build APK
flutter build apk

✨ Ekran Akışı

1. Login Screen → Email/Telefon/TC ile giriş
2. Search Screen → Otobüs seferi arama
3. Detail Screen → Sefer detayları ve harita
4. Seats Screen → Koltuk seçimi
5. Payment Screen → Ödeme özeti
6. Tracking Screen → PNR ve canlı konum takibi
